\section{Procedure}
\label{sec:c}

The response is a timed sequence.

\begin{enumerate}[leftmargin=2.0em,itemsep=3pt]
\item \textbf{Detect.}\quad the gate or a clinician flags the event and the robot enters a safe state
\item \textbf{Stabilize.}\quad the Assist role executes the immediate action within the contact-force budget
\item \textbf{Escalate.}\quad the Lead returns the interaction to a qualified human if the gate ESCALATES
\item \textbf{Report.}\quad the Reserve role files the adverse-event record within the required window
\end{enumerate}
